Creative Digital Experience Website

Files:
- index.html
- style.css
- script.js

How to run:
1. Extract the ZIP file.
2. Open index.html in any browser.
3. To publish online, upload these files to GitHub Pages, Netlify, or Vercel.

Main features:
- Dark futuristic cinematic design
- Neon particle canvas background
- 3D hover tilt cards
- Animated hero section
- Project showcase carousel
- Working contact form using localStorage
- Working download project plan button
- Responsive mobile menu
- Smooth scroll reveal animations

Edit guide:
- Change website name in index.html.
- Change colors in :root inside style.css.
- Change services/projects/contact text in index.html.
